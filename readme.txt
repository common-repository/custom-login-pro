Customize WordPress login page in minutes with Front-End Customizer.

== Description ==
Custom Login Pro is an advanced customizer which allows you edit WordPress login page in real-time. The plugin allows you **customize any element** in WordPress login page without coding. Adjust colors, fonts, sizes, positions and a lot more.

Custom Login Pro provides everything that you need for customizing your login page design. The plugin comes with advanced features like **visual resizing**, **drag & drop**, **measuring tool**, **google fonts** and a lot more.


=== Key Features ===
* Click and customize
* 50+ CSS properties
* Visual Drag & Drop
* Visual Margin & Padding Editing
* Live Element Resizing
* Manage the changes
* Undo / Redo

== Installation ==
1. Upload the plugin to your ‘wp-content/plugins’ directory, or download and install automatically through your admin panel.
2. Activate the plugin through the ‘Plugins’ menu in WordPress.
